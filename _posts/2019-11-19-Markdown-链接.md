## 链接

Markdown 支持导入链接。

<!-- more -->

语法规则：
```markdown
[链接名称](链接地址)
```
或者
```markdown
<链接地址>
```
使用实例：
```markdown
[Dunamist](https://github.com/dunamist)
```
或者
```markdown
<https://github.com/dunamist>
```
显示效果：

[Dunamist](https://github.com/dunamist)

或者

<https://github.com/dunamist>

### 高级链接

使用变量代替链接，并在文档末尾附上链接地址。语法规则：
```markdown
[链接名称1][1]

[链接名称2][链接地址]

[1]: https://github.com/dunamist
[链接地址]: https://github.com/dunamist
```
显示效果：

[链接名称1][1]

[链接名称2][链接地址]

[1]: https://github.com/dunamist
[链接地址]: https://github.com/dunamist
