## 代码

使用 \` 标记段落中的代码，代码前后各一个 \` 。

<!-- more -->

使用实例：
```markdown
a function of Python3: `print()`
```
显示效果：

a function of Python3: `print()`

---

使用 \` 标记一段代码，代码前后各三个 \`。语法规则：

\`\`\`key  
code segment  
\`\`\`

使用实例：

\`\`\`python  
import math

print(math.pi)  
\`\`\`

显示效果：

```python
import math

print(math.pi)
```

**注意**：在上述实例中，可以在第一行的三个 \` 之后，指定代码是何种语言，从而使代码高亮显示。当然也可以不指定。上述实例代码使用 Python3 编写，所以指定为 python。
