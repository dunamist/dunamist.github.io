## 代码

使用 \` 标记段落中的代码，代码前后各一个 \` 。

<!-- more -->

使用实例：
```markdown
a function of Python3: `print()`
```
显示效果：

a function of Python3: `print()`

---

使用 \` 标记一段代码，代码前后各三个 \`。

使用实例：

\`\`\`python  
import math

print(math.pi)  
\`\`\`

显示效果：

```python
import math

print(math.pi)
```

**注意**：Markdown 支持代码高亮显示。为开启代码高亮显示，需要在第一行的三个 \` 之后，指定代码是何种编程语言。上述代码使用 Python3 编写，因此指定为 python。
