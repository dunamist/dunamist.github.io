## 题目

输入两个整数，求这两个整数的最小公倍数。比如：14，16的最小公倍数是112。

<!-- more -->

## 代码

```python
# 使用辗转相除法求最大公约数
def gcd(x, y):
    if y > 0:
        return gcd(y, x % y)
    else:
        return x


# 通过 input() 得到的任何输入都是字符串类型，需要使用 int() 转化为整型类型
a = int(input())
b = int(input())

# 若第一个数不大于第二个数，则两数交换
if a <= b:
    a, b = b, a

# 最小公倍数 = a * b / 最大公约数
c = gcd(a, b)
print(int(a * b / c))

'''

    else:
```
